# LGPD - Registro de Solicitacoes (Modelo)

Data: 2026-04-01
Objetivo: registrar auditoria minima de atendimento.

Campos sugeridos:

- ID do ticket
- Data de recebimento
- Tipo de solicitacao
- Canal de entrada
- Responsavel pela verificacao
- Evidencias de identidade
- Data de conclusao
- Responsavel pela execucao tecnica
- Evidencias de execucao
- Data de resposta final
- Status final

Observacao:

Este registro deve ser armazenado em local seguro e acessivel para auditoria.
Registro operacional: Trello (quadro "LGPD BarberSaaS").
